

Pflegerouten ✔️
  - Routen erstellen ✔️
  - Pfleger Tagesroute festlegen ✔️
  - Pflegekunden zur Route hinzufügen ✔️ 
  - Route berechnen und für Pfelger speichern ✔️

Pflegerouten langfristig speichern ✔️
  - "Schichtplan" (Tagesrouten für alle Mitarbeiter im Schichtplan speichern) ✔️


Pflegekunden Pflegern zugeweisen  ✔️
  - Pflegekunden können einem oder mehreren Mitarbeitern zugewiesen werden ✔️
  - Später sehen Pfleger nur ihre zugewiesenen Kunden im Dashboard ✔️
  - Bei neuer Route für Mitarbeiter mit Kunden, die ihm nicht zugewiesen sind fragen, ob dies automatisch passieren soll ✔️




Pfleger Dashboard ✔️
  - Pfleger bekommt seine aktuelle Route angezeigt ✔️
  - Pfleger sieht eine Liste von allen ihm zugewiesenen Pflegekunden ✔️



Pfleger Pflegekunden Dashboard  ✔️
  - Pfleger bekommt ein Dashboard mit Kundendetails von ihm verwaltete Kunden ✔️
  - Hier hat er zudem eine Todo-Liste, was gemacht werden muss ✔️
  - Er kann bei jedem Besuch auf Besuch erstellen klicken. ✔️
  - Jeder Besuch bekommt automatisch die Standart Todos des Kundens aber es können auch manuell für bestimmte Besuche extra Todos hinzugefügt werden ✔️
  - Für jeden Besuch kann man dann die Todos automatisch abhacken ✔️
  - Und der Pfleger kann sich hier direkt für jeden Besuch Notizen machen (für spätere Dokumentation) ✔️
  - Im Admin Dashboard kann man die Besuche auch angelegt sehen sieht aber nur nach manueller Freigabe auch Todos und Notizen ✔️

Zugesicherte Leistungen für Kunde verwalten ✔️
  - Im Admin Panel kann man für jeden Kunden die vereinbarten Leistungen hinterlegen ✔️
  - Im Admin Panel können zudem Standart-Todos für einen Kunden eingetragen werden ✔️
  - Pfleger sehen für ihren Kunden alle Leistungen die zu erbringen sind ✔️
  - Pfleger sehen alle Standart-Todos, die aus dem Admin Panel kommen (können aber eigene hinzufügen, Standart-Todos jedoch nur nach Freigabe löschen oder updaten) ✔️


Routen Details ✔️
- Wenn Pfleger auf Route in der Home Page klickt, Details Seite öffnen ✔️ 
- Zeitplan (Zeitstrahl) mit allen Stopps und Uhrzeiten inklusive Start und Stopp ✔️
- Patienten werden bei den Stopps angezeigt, bei klickt, kommt man auf Patient Details Seite ✔️
- Route auf Karte anzeigen lassen ✔️


Dokumente für Kunden verwalten
- Im Admin Panel gibt es einen neuen Reiter "Dokumente"
- Hier können Verträge, Verordnungen, Angebote und sonstige Dokumente / Dateien hochgeladen werden
- Diese Dokumente können dort im Admin Panel dann CRUD verwaltet und gedownloaded werden
- Die Dokumente sind nur im Admin Panel sichtbar und nicht vom Pfleger einsehbar (DSGVO)

Mitarbeiter Dienstplan 
- Im Admin Panel neuer Punkt "Dienstplan"
- Hier kann der wöchentliche Dienstplan erstellt werden im Zusammenhang mit den Routen, die die Pfleger fahren müssen
- Der Dienstplan ist als Wochen-Tabellen Ansicht verfügbar
- Der Dienstplan kann als PDF oder Excel gedownloadet werden 
- Der Dienstplan kann wie der Routenplan global für jede Woche wiederholend erstellt werden und man kann einzelne Änderungen für bestimmte Tage festlegen
- Wenn im Dienstplan was geändert wird, soll automatisch geprüft werden, ob dann auch was im Routenplan geändert werden muss 
- Grundsätzlich wird der Dienstplan mit dem Routenplan genereiert
